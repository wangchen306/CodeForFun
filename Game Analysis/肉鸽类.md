# 肉鸽类游戏分析

## 杀戮尖塔

### 表象资料

* **美术风格**：
　　插画感的低饱和色调，人物与怪物造型夸张、富有辨识度，界面图标简洁、信息明确。视觉上既有独立游戏的个性，又保持卡牌与地牢题材的沉浸感。
* **故事剧情**：
　　没有长篇剧情驱动，更多是碎片化的背景与怪物笔记、成就描述，叙事以氛围与“探索残片”方式呈现，鼓励玩家自己拼凑世界观。
* **操作方法**：
　　鼠标点选卡牌、拖放出牌，动作直观；回合制战斗、无实时操作压力。
* **界面与系统**：
　　游戏信息显示在顶部状态栏，遗物显示在状态栏下方，不遮挡战斗；单位的能力则紧贴在血条下方显示；敌人的意图显示于其头顶，且不同意图对应不同图标，清晰明了；卡牌这一核心元素显示在底部中间，抽牌堆和弃牌堆分别显示在左下角和右下角，与抽弃牌的动画一致。各项信息的展示多而不乱。
* **游戏关卡**：
　　由多条分支构成的“爬塔”流程（节点地图），每局随机生成怪物组合、商店与事件；死亡则游戏失败，需从头开始。

### 感受资料

* **情绪**：
  + **兴奋与希望**：在获得一张核心卡牌或一个关键遗物时，会产生“这局稳了”的兴奋感。
  + **紧张与焦虑**：面对强大的精英敌人或Boss，每一步操作都如履薄冰，计算着伤害与格挡，充满了决策的压力。
  + **懊恼与不甘**：因一次失误、一次糟糕的抽牌（鬼抽）或一个错误决策而导致战损大大提升时，会感到强烈的挫败感。
  + **满足与顿悟**：成功用一套精心构筑的牌组战胜强敌，或者突然理解了某个卡牌组合的精妙之处时，会获得巨大的心流体验和智力上的满足感。这种“再来一局就能更好”的念头是游戏正反馈的核心。
* **难度**：
　　上手简单，精通困难。高进阶生存压力大，迫使玩家在短期过渡与长期收益之间抉择。
* **数值**：
　　游戏的一切都构建在精确的数值之上。伤害、格挡、能量、生命值、抽牌数等构成了每一回合的数学谜题。玩家的核心任务就是在有限的资源（能量、手牌）下，通过计算和权衡，做出最优的数值交换。值得一提的是，现版本的数值是制作组在大量的玩家反馈中不断调整而来的。
* **成就**：
　　除了通关和解锁内容的基础成就外，“进阶等级”本身就是游戏的一个重要成就。游戏也内置了46个成就，涵盖了各种不同的玩法和牌组构筑策略。

### 内在联系

* **形象**：
　　这里的“形象”并非指美术，而是指玩家在每一局游戏中构筑的牌组的战略形象或身份。这一局的你是“力量战”、“格挡战”，还是“无限流”？是“小刀贼”、“弃牌贼”，还是“毒贼”？这个“形象”是动态形成的，由玩家获得的卡牌和关键遗物共同决定。玩家需要根据随机获得的资源，灵活地为自己当前的这局游戏塑造一个清晰、有战斗力的“形象”，这是游戏策略深度的体现。
* **节奏**：
　　游戏包含微观和宏观两种节奏。
  + **微观节奏（战斗节奏）**：在单场战斗中，如何规划能量和手牌，何时爆发输出，何时全力防御，形成攻防转换的节奏。
  + **宏观节奏（爬塔节奏）**：在一整局游戏中，如何规划地图路线，决定战斗与休息的频率。
* **动机**：
　　驱动玩家一遍又一遍点击“开始游戏”的核心动机是：
  + **知识驱动的掌控感**：游戏的高度随机性看似不可控，但随着玩家对卡牌、遗物、怪物行为模式的理解加深，他们会发现自己掌控随机性的能力越来越强。每一次失败都像是一次付费学习，激励玩家带着新知识开启下一局。
  + **构筑的乐趣**：从零开始，将初始卡牌通过不断的选择、删除和升级，最终打造成一部高效、自洽、拥有强大协同效应的“杀戮机器”，这个过程本身就充满了创造的快感。
  + **永不重复的体验**：由于卡牌、遗物、地图和事件的高度随机组合，理论上每一局游戏都是独一无二的。这种永恒的新鲜感是Roguelike游戏的核心魅力。

### 基础设定

《杀戮尖塔》的伟大之处在于，它将牌组构筑从一个相对小众的桌面游戏机制，与成熟的Roguelike电子游戏框架完美结合，创造出了一种全新的、极易上瘾的游戏循环。

* **题材**：
　　游戏属于黑暗奇幻题材，但如前所述，题材和故事仅为玩法提供了一个简洁的背景。
* **玩法**：
　　核心玩法是“肉鸽（Roguelike）”与“牌库构筑（Deck-Building Game, DBG）”的融合。
  + **肉鸽（Roguelike）**：体现在三个核心要素上：随机生成（地图、奖励）、永久死亡（失败则从头开始）和高重复可玩性。
  + **牌库构筑（DBG）**：与《万智牌》或《炉石传说》等集换式卡牌游戏（CCG）不同，玩家不是在游戏外预先组好一副牌，而是在游戏过程中动态地构筑自己的牌库。这使得每一局的策略都必须因地制宜，而不是固守一套卡组。

### 商业目标

《杀戮尖塔》出现之前，数字卡牌游戏市场由《炉石传说》等重度依赖开包和持续付费的CCG主导，而Roguelike游戏则以《以撒的结合》、《挺进地牢》等动作类游戏为主。

《杀戮尖塔》精准地切入了一个几乎空白的市场——买断制的单人PVE卡牌策略游戏。它融合了两个热门类型的优点，却没有它们的商业化弊端。

它不仅是一款成功的游戏，更是“肉鸽卡牌”这一子类型的定义者和标杆。在其之后涌现了大量模仿者（如《怪物火车》、《月圆之夜》等），但鲜有能超越其平衡性和设计深度。

* **用户画像**：
  + **核心用户**：策略游戏爱好者、棋牌游戏玩家、传统Roguelike游戏玩家。他们享受深度思考、智力挑战，并对高难度有追求。
  + **拓展用户**：厌倦了CCG“氪金”模式的卡牌游戏玩家；寻找能够提供“再来一局”短平快体验的玩家；以及喜欢通过学习和积累知识来获得成长的玩家。
  + **特点**：这群用户通常不追求顶级的画面表现，而更看重游戏的核心玩法、策略深度和重复可玩性。他们是“为玩法付费”的典型代表。
